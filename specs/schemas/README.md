# JSON Schemas

Draft JSON Schema 2020-12 contracts for Sphere protocol primitives.

Current schemas:

- `entity.schema.json`
- `identity-link.schema.json`
- `edge.schema.json`
- `event.schema.json`
- `command.schema.json`

These schemas are unstable until a protocol version is declared stable. Current draft schema IDs use the `/draft/0.1.0/` path and require `schemaVersion: "0.1.0"` on records. See [Protocol and schema versioning](../../docs/protocol-versioning.md).
