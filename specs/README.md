# Specs

Draft language-agnostic Sphere protocol materials.

Initial areas:

- `schemas/` — JSON Schema contracts.
- `examples/` — valid example objects.
- `test-vectors/` — valid/invalid fixtures and hash-chain vectors.
- `draft/` — explanatory draft specs before extraction to a future `sphere-protocol` repo.

Current draft protocol/schema version: `0.1.0`. See [Protocol and schema versioning](../docs/protocol-versioning.md) for the compatibility policy and version bump checklist.

Everything here is unstable until explicitly versioned as stable.
